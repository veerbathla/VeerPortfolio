# VeerPortfolio
# 🚀 Personal Portfolio Website

A modern and minimal portfolio website built to showcase my projects, skills, and experience as a full-stack web developer.

---

## 🧑‍💻 About the Project

This portfolio is designed with a focus on simplicity, performance, and clean UI.
It highlights my work, technical skills, and provides an easy way to get in touch.

---

## ⚙️ Tech Stack

* **Frontend:** React.js, Tailwind CSS, DaisyUI
* **Backend / Services:** EmailJS (for contact form)
* **Routing:** React Router
* **Other Tools:** Vite

---

## ✨ Features

* Clean and minimal UI
* Fully responsive design
* Projects showcase section
* Tech stack overview
* Contact form with EmailJS integration
* Direct links to social profiles (GitHub, LinkedIn, LeetCode)
